# We-Study Timer

this is a simple pomodoro timer made using html, css and javascript

it helps in managing study or work sessions by using focus and break cycles

---

## features

- 25 minute focus timer  
- 5 minute break timer  
- start, pause and reset controls  
- automatic switching between focus and break  
- session counter to track completed cycles  
- sound alert when timer finishes  

---

## how to use

1. click start to begin the timer  
2. pause if needed  
3. reset to restart  
4. switch between focus and break modes  
5. complete sessions and track your progress
